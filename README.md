# Bavatharanicodeit3.0

Astronaut Fitness Tracker is a JavaScript-based web application that allows users to track exercises, log nutrition, monitor vital signs, and simulate daily movement through a space-themed, interactive interface.

This project was created by **Bavatharani**, a Grade 10 student, for the **CODE IT 3.0** competition conducted by **eduSeed**, showcasing her creativity, effort, and interest in coding and UI design.
